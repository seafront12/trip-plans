# 崇左家庭亲子游完整攻略

**出发城市：** 香港
**目的地：** 广西崇左市
**旅行日期：** 2026年4月3日 — 4月8日（共6天5晚）
**家庭成员：** 2名成人 + 2名儿童（9岁、11岁）

---

## 三、推荐交通方案

### 推荐选择：高铁 G418 去程 + G417 回程

**理由：**
- 全程直达，香港西九龙 → 南宁东，无需转机或候机，儿童乘坐舒适
- 价格比机票更实惠，一家四口合计约CNY 1,860（约HKD 2,000），明显低于机票方案
- 班次固定（每日一班），计划简单
- 高铁座位宽敞，有餐车，适合家庭出行

**预订建议：**
- 中国铁路官网 [12306.cn](https://www.12306.cn/en/) 或携程提前购票（建议出发前2-4周购票）
- 需护照信息购票，入境中国须持有效中国签注（回乡证或BNO持有人需签证）

## 六、每日详细行程

---

### 第1天：4月3日（周五）—— 出发·抵达崇左

```
09:00  早餐 + 收拾行李，前往香港西九龙高铁站
11:00  抵达香港西九龙站，办理乘车手续（建议提前1.5小时到达）
       ⚠️ 须携带护照 + 中国签注/回乡证，高铁实名制
15:33  乘坐 G418 高铁出发，香港西九龙 → 南宁东
       - 二等座，全程约4小时，车上可订餐
       - 沿途经过广州、佛山等城市，孩子可观景
17:00  车上简餐 / 自带零食
19:32  抵达南宁东站
19:45  购买 南宁东 → 崇左南 高铁票（约28分钟，CNY 20/人）
20:20  抵达崇左南站，出租车/滴滴打车前往酒店（约15分钟）
20:40  入住酒店（建议住崇左市区，如友谊大道附近）
21:00  附近餐厅晚餐（推荐：崇左本地壮族风味餐厅）
       - 必点：酸笋牛腩、五色糯米饭、南宁老友粉
22:00  休息、准备次日游览
```

**住宿推荐：** 崇左市区（江州区），优先选择有家庭房的连锁酒店（如汉庭、如家、维也纳），预算CNY 300–500/晚。

```
08:30  酒店早餐
```

```
10:00  出发前往大新县明仕田园（约120km，约1.5–2小时）

       推荐：景区附近农家乐，农家菜 + 河鱼
13:00  进入明仕田园景区（免费入场！）
       🚲 租单车：约CNY 50/辆/小时
13:30  骑行稻田乡间小道（约1.5小时）
       - 路线：穿越峰丛谷地、稻田、壮族村落
```

## 八、实用贴士

### 天气与穿着
- **4月崇左天气**：气温约22–30°C，属亚热带季风气候，较为潮湿
- **降雨**：4月为春雨季节，建议携带轻便雨衣或折叠伞（非雨伞，方便游览）
- **穿着**：轻薄长裤（防虫防晒）+ 防滑运动鞋，德天瀑布道路有水雾，需防滑
- **防晒**：白天紫外线强，防晒霜+帽子必备

### 4月特别提醒：德天瀑布水量
- 4月属于旱季末期，瀑布水量低于旺季（6–10月），但仍然壮观
- 若孩子对大瀑布期望较高，提前告知"旱季瀑布也很美"

### 支付方式
- **主要支付**：微信支付、支付宝（香港居民可绑定港卡开通）
- **现金**：建议携带CNY 1,000–1,500现金备用，偏远地区小店可能不支持电子支付
- **银行卡**：Visa/Mastercard 在大商场和酒店可用，小景区不一定接受

### 本地交通
- **打车**：推荐使用高德地图（内置打车功能）或滴滴出行，比路边拦车安全且便宜
- **租车/包车**：前往大新、宁明景区强烈建议提前一天联系当地旅行社包车，含司机约CNY 400–600/天，省去驾照问题
- **崇左南站旅游直通车**：崇左旅游集散中心有旅游专线直达明仕田园和德天，每日09:15出发，价格更优惠

### 入境与签注
- **香港永久居民（持回乡证）**：凭回乡证可直接入境内地，无需另外签证
- **外籍人士**：需提前申请中国签证
- **高铁出入境**：香港西九龙站设有"一地两检"，在香港办完内地出境手续

### 亲子出行贴士
- 带好常备药：止泻药、防蚊喷雾、创可贴
- 德天瀑布区域有水雾，备一套换洗衣物
- 孩子在竹筏漂流时须穿救生衣（景区提供）
- 花山岩画游船须遵守安全规定，勿让孩子靠出扶手栏外
- 行程安排留有弹性，孩子累了可随时调整

### 通讯
- 购买内地SIM卡（到达南宁东站或崇左即可购买）或开通香港漫游套餐
- 推荐App：高德地图（导航）、大众点评（餐厅）、携程/去哪儿（门票预订）、微信（支付）
